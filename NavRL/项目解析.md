### isaac-training/ - 训练根目录
核心训练模块，基于NVIDIA Isaac Sim构建的强化学习训练环境
###### 1. setup.sh - 完整训练环境安装脚本
###### 2. setup_deployment.sh - 轻量级部署环境脚本

### third_party/ - 第三方依赖库
包含训练所需的所有核心框架和库：
###### 1.OmniDrones/ - 无人机仿真框架
```
基于Isaac Sim的无人机仿真环境
提供多旋翼无人机模型和控制器
包含环境配置、文档、示例等
```
###### 2.orbit/ - NVIDIA Isaac机器人学习框架
```
提供GPU加速的物理仿真
支持大规模并行训练
包含传感器、地形生成等模块
```
###### 3.rl/ - TorchRL强化学习库
```
基于PyTorch的RL框架
提供PPO等算法实现
包含数据收集、环境封装等工具
```
###### 4.tensordict/ - 张量字典库
```
高效的张量数据结构
支持批量操作和GPU加速
RL数据处理的核心组件
```
###### 5.warp/ - GPU加速计算库
```
NVIDIA的GPU计算框架
提供高性能物理仿真
支持CUDA加速
```

###  training/ - 核心训练代码
#### 1.cfg/ - 配置文件目录
```yaml
# train.yaml - 主训练配置
max_frame_num: 12e8        # 最大训练帧数
env:
  num_envs: 2              # 并行环境数量
  num_obstacles: 350       # 静态障碍物数量
env_dyn:
  num_obstacles: 80        # 动态障碍物数量
wandb:                     # 实验跟踪配置
  project: NavRL
  ```

  ```yaml
  #drone.yaml - 无人机配置
drone:
  model_name: "hummingbird"  # 蜂鸟无人机模型
sensor:
  lidar_range: 4.0          # 激光雷达探测范围
  lidar_vbeams: 4           # 垂直激光束数量
  ```

  ```yaml
  # ppo.yaml - PPO算法参数
  algo:
  actor:
    learning_rate: 5e-4     # 策略网络学习率
    action_limit: 2.0       # 动作限制(m/s)
  critic:
    learning_rate: 5e-4     # 价值网络学习率
```

```yaml
#sim.yaml - 仿真参数
sim:
  dt: 0.016                 # 仿真时间步长
  use_gpu_pipeline: true    # 启用GPU加速
  gravity: [0, 0, -9.81]    # 重力设置
```
